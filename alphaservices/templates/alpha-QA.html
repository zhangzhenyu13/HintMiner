Programming Alpha is a machine question-asnwering AI system. Programming Alpha QA service

The service currently support AI related questions answering for relative engineers and learners. The system backend is based on large corpus information aggregation. currently ,the knowledge data base is extracted from:

  • Stack Overflow
  • Crossvalidated
  • Artificial Intelligence
  • Data Science
besides, we are now gathering AI related corpus from wikipedia to extend the current knowledge base.(we lack fundation support to rent ips which can boost the crawlers significantly!!!)
The core part of the system is a ngram hashing(murmurhash3) retrieval model and a multi-task fine-tuning Semantic Understanding Model which shares some weights of google Bert Base.
